# References

This folder keeps small project reference files:

- `projects.pdf`: original course project brief.
- `SUSTech_RAG_工程实施交接手册.pdf`: implementation handoff document used during planning.
- `config_screenshot.png`: environment/configuration screenshot.

Runtime data, indexes, evaluation outputs, and model weights are intentionally not
stored here. They are regenerated or downloaded by the scripts in `sustech-rag/`.
